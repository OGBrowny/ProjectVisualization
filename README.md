📊 2D & 3D Data Visualization using Python

📌 Overview

This project demonstrates data visualization techniques using Python.
It includes both 2D and 3D graphs created using different datasets.

---

🛠️ Tools & Libraries Used

- Pandas → Data handling and analysis
- NumPy → Numerical computations and data generation
- Matplotlib → Data visualization (2D & 3D graphs)

---

📂 Datasets Used

1. CSV Dataset ("train.csv")
   
   - Used for basic visualizations
   - Contains fields like Category, Sales, Order Date

2. NumPy Generated Data
   
   - Used for mathematical and special plots
   - Created using functions like "linspace()"

3. Meshgrid Data
   
   - Used for 3D surface-based visualizations

---

📈 Visualizations Included

🔹 2D Graphs (CSV Data)

- Bar Graph → Sales by Category
- Pie Chart → Sales Distribution
- Line Graph → Sales Over Time
- Histogram → Sales Distribution

🔹 Special Plots (NumPy Data)

- Plot (Line Graph)
- Scatter Plot
- Bar Graph (NumPy Data)
- Log-Log Plot
- Semi-log X Plot
- Semi-log Y Plot

🔹 Subplots

- Multiple graphs displayed in a single figure

🔹 3D Visualizations

- 3D Scatter Plot
- Surface Plot
- Wireframe Plot
- Contour Plot

---

▶️ How to Run

1. Make sure Python is installed
2. Install required libraries:
   pip install pandas numpy matplotlib
3. Place "train.csv" in the same folder as "project.py"
4. Run the file:
   python project.py

---

🎯 Conclusion

This project shows how Python can be used to visualize data effectively using different types of graphs.
Using multiple datasets improves understanding and provides better insights.
